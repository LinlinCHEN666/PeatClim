This folder is intentionally distributed without WorldClim raw data.

This project requires WorldClim version 2.1 bioclimatic variables
at 5 minutes resolution (1970-2000 average).

Download from:
https://www.worldclim.org/data/worldclim21.html

After downloading, extract the bioclimatic GeoTIFF files into:
data_external/WorldClim/wc2.1_5m_bio/

See the main project README.md for the expected directory structure and citation details.
